# Dads Can't Beatbox

**But they're gonna try anyway.**

A drag-and-drop music production toy where dad characters each get a sound loop - drums, bass, hi-hats, guitar, vocals, synths, and more. Stack loops, mute the off-beat dad, solo the one who's actually vibing, and somehow end up with a track that slaps harder than a minivan subwoofer at school pickup.

**[beatbox.jubinsanghvi.com](https://beatbox.jubinsanghvi.com)**

## What Is This

Every dad thinks they can beatbox. Most can't. This app lets them live the dream anyway - 40 loops across 10 categories, all locked to 92 BPM, all perfectly synced so even the most rhythmically challenged father can sound like a producer.

Drag a sound onto a dad. Hit play. Pretend you made it on purpose.

## Features

- **Drag & drop** sound loops onto performer slots
- **Mute / solo** individual dads (sorry, Gary)
- **10 sound categories** - Drums, Bass, Hi-Hat, Guitar, Perc, Vocals, FX, Synth, Pads, Risers
- **Export to WAV** - download your masterpiece
- **Share via URL** - send your mix to other dads
- **Equalizer visualizer** - watch the bars bounce
- **Dark mode** - for late-night dad sessions after the kids are asleep

## Run Locally

```bash
npm install
npm run dev
```
